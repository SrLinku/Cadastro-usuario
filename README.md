# Cadastro de Usuários

Aplicativo Android simples para cadastro de usuários, desenvolvido com o objetivo de praticar conceitos de desenvolvimento mobile utilizando **Android Studio**.

O projeto utiliza **Java para a lógica do aplicativo** e **XML para a construção das interfaces gráficas**.

---

## 📌 Objetivo

Desenvolver um aplicativo básico de cadastro de usuários para praticar conceitos como:

* Desenvolvimento de aplicativos Android
* Estrutura de projetos no Android Studio
* Programação orientada a objetos com Java
* Criação de interfaces utilizando XML

---

## 🚀 Funcionalidades atuais

* Tela de cadastro de usuário
* Estrutura inicial de classes em Java
* Interface construída em XML
* Estrutura básica de aplicativo Android

---

## 🛠 Tecnologias utilizadas

* **Java** – lógica do aplicativo
* **XML** – criação das interfaces
* **Android Studio** – ambiente de desenvolvimento
* **Android SDK** – ferramentas de desenvolvimento Android
* **Git** – controle de versão
* **GitHub** – hospedagem do projeto

---

## 📂 Estrutura do projeto

Cadastro_usuario
│
├── java/ → classes do aplicativo
├── res/layout/ → arquivos XML das interfaces

└── demais arquivos do projeto Android

---

## ▶️ Como executar

1. Clone o repositório:

git clone https://github.com/SrLinku/Cadastro-usuario.git

2. Abra o projeto no **Android Studio**

3. Aguarde o **Gradle sincronizar**

4. Execute o projeto em:

* um **emulador Android**
* ou um **dispositivo físico**
