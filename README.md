# 📱 Cadastro de Usuários - Android

Aplicativo Android simples para **cadastro e visualização de usuários**, desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento mobile utilizando **Android Studio**.

O projeto utiliza **Java para a lógica da aplicação** e **XML para construção das interfaces**.

---

# 📌 Objetivo do Projeto

Este projeto foi criado com foco em aprendizado de desenvolvimento Android, abordando conceitos como:

* Estrutura de projetos Android
* Programação orientada a objetos com Java
* Criação de interfaces utilizando XML
* Manipulação de listas com RecyclerView
* Navegação entre telas (Activities)

---

# 🚀 Funcionalidades atuais

✔ Exibição de usuários cadastrados em uma lista
✔ Interface construída em XML
✔ Uso de RecyclerView para renderização da lista
✔ Adapter para gerenciamento dos dados exibidos
✔ Navegação da tela principal para tela de cadastro
✔ Atualização automática da lista ao retornar para a tela principal

---

# 🛠 Tecnologias utilizadas

* **Java** → lógica do aplicativo
* **XML** → layout das interfaces
* **Android Studio** → ambiente de desenvolvimento
* **Android SDK** → ferramentas de desenvolvimento Android
* **RecyclerView** → exibição de listas dinâmicas
* **Git e GitHub** → controle de versão

# ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/SrLinku/Cadastro-usuario.git
```

2. Abra o projeto no **Android Studio**

3. Aguarde o **Gradle sincronizar**

4. Execute o aplicativo em:

* um **emulador Android**
* ou um **dispositivo físico**
